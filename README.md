# 🍞 The Daily Loaf - Artisan Bakery Website

## Project Overview

The Daily Loaf is a fully responsive, multi-page website for an artisan bakery based in Christchurch, New Zealand. The website showcases the bakery's products, tells its story, and provides functionality for customers to enquire about catering, special orders, and contact the bakery.
